A Note to OS X 10.10 beta Users

Code signing and umlauts (Ü) don't go well together on the current beta of Yosemite. This means, Übersicht will crash with a 'invalid signature' error. However, you can rename the app to Uebersicht, or another name that contains only ascii characters, and it will work.
